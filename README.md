# 💰 FinConsult — Consultoria Financeira Inteligente

Plataforma completa de consultoria financeira com 5 módulos integrados:

- **👤 Cadastro do Cliente** — Dados pessoais, metas e importação via CSV
- **📊 Relatório Mensal** — Receitas, despesas, dívidas e investimentos
- **🧮 Simulador de Dívidas** — Estratégias Avalanche, Bola de Neve e Proporcional
- **📒 Controle Financeiro** — Lançamentos diários com categorização
- **📈 Dashboard** — Score de saúde, KPIs, gráficos e resumo executivo

## 🚀 Como rodar localmente

```bash
npm install
npm run dev
```

## 📦 Tech Stack

- React 18
- Recharts (gráficos)
- Vite (build)
- GitHub Pages (deploy)

## 🌐 Deploy

O deploy é automático via GitHub Actions. A cada push na branch `main`, o app é publicado em:

```
https://SEU-USUARIO.github.io/finconsult-app/
```

### Configuração do GitHub Pages:
1. Vá em **Settings → Pages**
2. Em **Source**, selecione **GitHub Actions**
3. Pronto! O workflow já está configurado.
